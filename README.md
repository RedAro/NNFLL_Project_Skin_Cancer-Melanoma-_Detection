# Skin Cancer (Melanoma) Detection Using Deep Learning

This project uses CNN, VGG16, and MobileNetV2 models to classify dermoscopic images into benign and malignant melanoma.

## Dataset
- HAM10000 dataset
- 10,000+ images â†’ Subsampled to 1,200 balanced images

## Models Used
- Custom CNN
- VGG16 (Transfer Learning)
- MobileNetV2 (Lightweight)

## Results
- VGG16: **87% accuracy**
- CNN: **84% accuracy**
- MobileNetV2: **82% accuracy**
